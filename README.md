# Tirek — Kaspi анти-демпинг и калькулятор реальной маржи

Расширение Chrome для селлеров Kaspi.kz: реальная маржа по товару (комиссии Kaspi 2026 + налоги РК) и анти-демпинг конкурентов прямо на карточке. Бесплатно калькулятор и 3 товара под наблюдением, дальше Pro. Локально, без регистрации.

**Закрытый альфа-тест, 0.1.0-alpha.10.** Manifest V3 + TypeScript + React 19. Светлая тёплая тема (кремовый фон, оранжевый акцент).

📄 [Политика приватности](https://yerkebulan-ardabayev.github.io/Tirek-Extension/privacy.html)

## Для тестеров → [landing-page](https://yerkebulan-ardabayev.github.io/Tirek-Extension/)

Скачайте zip из [релизов](https://github.com/Yerkebulan-Ardabayev/Tirek-Extension/releases/latest), распакуйте, `chrome://extensions` → Developer mode → «Загрузить распакованное». Баги и идеи: [Issues](https://github.com/Yerkebulan-Ardabayev/Tirek-Extension/issues).

## Для разработчиков

```bash
pnpm install
pnpm test       # vitest
pnpm typecheck  # tsc --noEmit
pnpm build      # → dist/ для Load Unpacked
pnpm package    # build + zip → releases/
```

Dev-билд: `pnpm build`, затем `chrome://extensions` → Developer mode → «Загрузить распакованное» → папка `dist/`.

## Что делает

- **Overlay на карточке товара** (`kaspi.kz/shop/p/*`) — бейдж в углу: сколько демперов ниже вашей цены или что вы дешевле всех. Клик открывает панель со всеми продавцами и кнопкой «Следить».
- **Калькулятор маржи** (popup, вкладка «Маржа») — чистая прибыль с учётом комиссий Kaspi, НДС, эквайринга, Red, СПП, доставки, рекламы, возвратов, закупки и налога.
- **Обзор магазина** — все ваши товары в одной таблице: цена, комиссия, налог, остаток, демперы, прибыль. Себестоимость вводится файлом, вставкой или вручную.
- **Наблюдение + уведомления** — watchlist по SKU, push при новом демпере на отслеживаемом товаре.
- **Тариф** — бесплатно калькулятор + 3 товара; Pro 2990 ₸/мес безлимит. Оплата вне расширения, активация офлайн-кодом (ECDSA P-256, привязан к установке).

## Тарифы и налоги

Комиссии Kaspi и налоги РК на 2026 верифицированы по первоисточникам (adilet.zan.kz, kgd.gov.kz, enpf.kz). Конкретные ставки и ссылки на источники живут в коде: `src/lib/kaspi-fees.ts` и `src/lib/kz-taxes.ts` (там же ставка региональной упрощёнки, настраиваемая селлером).

## Структура

```
src/lib/            ядро (расчёты, парсеры, налоги) — покрыто тестами
src/content/        overlay + парсер карточки товара (Shadow DOM)
src/popup/          React-попап (маржа, демперы, наблюдение, настройки)
src/store-overview/ страница «Обзор магазина»
src/background/     MV3 service worker (уведомления, телеметрия)
scripts/build.mjs   esbuild + генератор иконок
```

## Если Kaspi поменяет DOM

Парсеры используют списки fallback-селекторов и логируют сработавший в консоль (`[Tirek] ...`). При поломке откройте DevTools на странице товара и добавьте новый селектор в начало массива в `src/lib/kaspi-shop-parser.ts`.
